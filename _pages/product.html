Mobile Interface

About the Project

This is a mobile interface I had designed during Code Wars 2016. Code Wars is the annual interschool IT Fest organised by Delhi Public School, Vasant Kunj.

We were a team of 3, and the task was to create a 3D model, a video and a user interface for a phone.
I was incharge of making the User Interface for the phone's operating system.
